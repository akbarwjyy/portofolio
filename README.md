# Portfolio Website

Modern portfolio website, built with Next.js and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS
- **Font:** Inter & Space Grotesk
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
